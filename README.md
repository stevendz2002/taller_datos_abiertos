# 🇨🇴 Explorador API Colombia  
### Taller de Datos Abiertos con Flutter

Aplicación móvil desarrollada en **Flutter** que consume la API pública **API Colombia**, permitiendo explorar información histórica, cultural y geográfica del país mediante una interfaz moderna y organizada.

El proyecto implementa arquitectura modular, consumo de REST API, manejo de modelos JSON, navegación con go_router y componentes reutilizables.

---

# 🎯 Descripción de la API

La aplicación consume datos desde la API pública:

🔗 https://api-colombia.com/

El endpoint base utilizado es:
https://api-colombia.com/api/v1/

Se consumen los siguientes endpoints:

| Endpoint | Descripción |
|--------|-------------|
| `/President` | Presidentes de Colombia |
| `/TypicalDish` | Platos típicos del país |
| `/Department` | Información de departamentos |
| `/Airport` | Aeropuertos de Colombia |
